## 📱 Telegram通知系统

### 什么是Telegram通知？
网站会自动发送消息到您的Telegram，告诉您：
- 每天新文章发布成功
- 网站部署状态
- 系统错误警告

### 正常的通知内容
**成功通知示例**:
```
📝 新内容发布 | 08-31 09:15

📄 最佳智能插座推荐2025...
🏷️ Smart Plugs
📊 2847 字
🔍 smart plug, alexa

🌐 查看网站
🤖 自动化内容生成
```

**构建成功通知示例**:
```
✅ 构建成功 | 08-31 09:20

📝 自动更新
⏱️ 45秒
🔗 abc1234

🌐 查看网站
🤖 GitHub Actions
```

### 异常通知处理
**如果收到错误通知**:
```
🚨 系统错误 | 08-31 09:15

🔸 类型: 内容生成失败
🔸 错误: API限制超出
🔸 环境: 生产环境

⚠️ 需要人工检查
🤖 自动错误监控
```

**处理步骤**:
1. 不要慌张，大部分错误会自动恢复
2. 等待24小时，看第二天是否恢复正常
3. 如果连续2天出错，需要技术支持

## 💰 收益和商业化管理

### Google AdSense管理

#### 当前状态
- **申请状态**: ❌ 待申请 (图片匹配问题需要先解决)
- **预计月收入**: $100-300 (基于预估流量)

#### 如何申请AdSense
1. **准备工作** (必须完成):
   - 网站必须使用真实产品图片 (不能是占位符)
   - 网站内容丰富 (至少30篇文章)
   - 网站访问正常，无错误页面

2. **申请步骤**:
   - 访问 https://adsense.google.com 
   - 添加网站地址: https://ai-smarthome.vercel.app
   - 完成税务信息填写
   - 等待审核 (通常7-14天)

3. **审核后**:
   - 通过后会收到邮件通知
   - 网站会自动开始显示广告
   - 开始产生广告收益

## 🚨 常见问题解决

### 网站无法访问
**症状**: 打开ai-smarthome.vercel.app出现错误
**解决步骤**:
1. 等待5-10分钟再试 (可能是临时问题)
2. 检查网络连接是否正常
3. 尝试用手机流量访问
4. 如果持续无法访问，查看Telegram是否有错误通知

### Telegram没有收到通知
**症状**: 超过24小时没收到任何Telegram消息
**解决步骤**:
1. 检查Telegram是否正常工作 (发送其他消息测试)
2. 确认没有屏蔽机器人消息
3. 等待下一次自动运行 (每日上午9点)

### 网站没有新文章
**症状**: 连续几天没有新文章发布
**解决步骤**:
1. 检查Telegram通知，看是否有错误消息
2. 等待24-48小时 (API限制会自动恢复)
3. 如果超过48小时仍无新文章，需要技术检查

### 图片显示不正常
**症状**: 文章中的图片无法显示或显示错误
**解决步骤**:
1. 刷新页面重新加载
2. 检查网络连接
3. 等待下次网站更新 (图片会自动修复)
